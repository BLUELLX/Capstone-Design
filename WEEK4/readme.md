# 4주차 기획서 - 로드맵 데이터 및 사용자 선택 기능 설계

## 1. 주차 목표

- 목표별 로드맵 데이터 구조 설계
- 사용자 목표 선택 기능 구현 (기본 동작 확인)
- 선택된 목표에 따라 동적 페이지 이동 준비
- 체크리스트 UI 기본 틀 구성 및 기능 연결 계획 수립

---

## 2. 로드맵 데이터 설계

사용자가 선택할 수 있는 진로/관심 분야별로  
단계별 자기계발 항목을 배열 형태로 구성함.

예시 (JSON 형태로 저장 예정):

```json
{
  "frontend": {
    "title": "프론트엔드 개발자",
    "steps": [
      "HTML/CSS 기본 문법 익히기",
      "JavaScript 문법과 DOM 이해",
      "React 기초 문법 익히기",
      "작은 프로젝트로 연습하기",
      "포트폴리오 제작"
    ]
  },
  "ux": {
    "title": "UX 디자이너",
    "steps": [
      "디자인 툴(Figma 등) 사용법 익히기",
      "UX 리서치 기초 공부",
      "와이어프레임 및 프로토타입 제작",
      "사용자 피드백 반영하기",
      "포트폴리오 정리"
    ]
  }
}
